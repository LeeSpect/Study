N = 문제에서 주어진 input 값의 범위

* N <= 10   
-> O(N1), O(2^N), O(3^N)

* N <= 20   
-> O(2^N)

* N <= ```100```   
-> O(N^4)  == **4중** for문까지

* N <= ```500```   
-> O(N^3)  == **3중** for문까지

* N <= ```1000```   
-> O(N^2), O(N^2 * logN)  = **2중** for문까지   

* N <= ```100,000```   
-> O(N), O(NlogN), O(logN), O(1)
